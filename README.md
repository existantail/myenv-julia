# Julia Environment: myenv

This repository contains a reproducible Julia environment using:

- Project.toml
- Manifest.toml

It is designed to ensure consistent package versions across different servers.

---

## Purpose

This environment is used for:
- Optimization (JuMP)
- Data analysis (DataFrames)
- Statistics & modeling (Distributions, StatsModels)
- Solvers (NLsolve, NLopt)
- Python/R interoperability (PyCall, RCall)
